My Coding Story

I'm Miles, a Manchester based homebody and free spirit. I love to immerse myself in the depths of code into the late hours of night, forgetting about the real world.

I enjoy resolving programming problems & designing captivating user interfaces in order to develop rich web experiences. I like to be faced with a challenge to learn new skills during the development of projects.

Recently I've gained valuable experience while creating several websites as a freelancer, meeting each of the client's unique end objectives and design requirements. I've also been programming mods/minigames in Javascript, for a private server in GTA Online giving players additional challenges, adding more fun to gameplay and making the server unique. I'm currently building up my knowledge in Typescript and React.js.

As well as being a self-taught programmer I graduated from IT college in 2011 where I learnt the fundamentals of programming languages and computer science.

I am looking forward to expanding my career in web development and I'm highly interested in learning backend technologies in the near future.
